# Conclusion
I am proud to say that I have successfully developed an application using ASP.NET Core and Blazor, with a focus on nested entities and responsive design. I utilized the power and flexibility of the ASP.NET Core framework to build a robust web application, and used Blazor to create interactive and dynamic user interfaces using C#. I also paid attention to responsive design to ensure that my application was accessible and easy to use on any screen size. Overall, this was a challenging but rewarding project, and I am pleased with the results.
